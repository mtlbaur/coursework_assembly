This program creates a movable and resizable box that contains ASCII characters and their corresponding decimal and hexadecimal codes.
The contents of this box can be scrolled by pressing the up and down arrows keys.
The box can be resized by holding the ctrl key and pressing either the up or down arrow key.
The color of the box can be cycled via left alt.
The box can be moved via the w, a, s, and d keys.
The program can be terminated via esc.
